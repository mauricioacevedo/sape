MONITOREO DE UNA LINEA TELEFONICA

 

Esta opción permite al SAPE realizar un seguimiento a determinada línea telefónica. Con ella se podrá interactuar con el usuario de la línea en caso que esto se requiera, como en una comunicación tripartita. Para realizar el monitoreo hay que hacer lo siguiente:

  • Digite el numero telefónico en el campo “Teléfono”, de la opción “Pruebas Telefónicas”
  • Luego haga Click en “Monitoreo”, y se abrirá otra ventana.
  • Ingrese el numero de teléfono desde donde se realizará el monitoreo. (Este numero telefónico no es el numero al cual se le esta haciendo la prueba si no por donde se escuchará a la línea).

 

  • En caso de que el cabezal se SiplexPro o Indigo, podrá seleccionar el tipo de monitoreo que desea realizar.

 

En las opciones que se pueden ver en la ventana que se abre para las cabezas SiplexPro e Indigos se encuentran las siguientes:

MONITOREAR: Esta opción permite hacer un seguimiento del estado de la línea que desea, en caso de que la línea se encuentre ocupada entonces se escuchara la conversación que en ese momento este teniendo el usuario de la misma.  Dentro de esta opción también se habilita el campo Dial. Con Dial el usuario del SAPE podrá realizar una llamada desde su PC a un numero telefónico cualquiera, esta llamada permite verificar si desde la línea que se esta monitoreando se pueden realizar llamadas. La llamada saldrá como una llamada realizada desde el teléfono que se esta monitoreando pero no se podrá establecer conversación alguna.

INTERACTUAR: Aquí el usuario del SAPE puede hacer una llamada a la línea que desea probar, si la línea no esta en uso la llamada entrara como una llamada normal, en caso de que la línea esta ocupada por el usuario entonces la llamada entrara como una llamada tripartita. Si se selecciona la opción interactuar se habilita el campo Test Dtmf.  Test Dtmf trae a la pantalla las teclas que el usuario de la línea a la cual se esta probando este marcando en ese momento.

TRACER: Esta opción permite al usuario del SAPE generar un ruido en el armario al cual pertenece la línea telefónica, en caso de que la persona encargada de reparar dicha línea no la encuentre dentro de este.

HOWLER: Esta prueba genera una sirena en la línea telefónica que se este probando para alertar al usuario de que tiene el teléfono descolgado.
